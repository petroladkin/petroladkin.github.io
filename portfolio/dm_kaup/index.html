Kaup

Application about the Viking village Kaup

Unity | C#

iOS | Android

Development description

  • Designed and developed the product based on screens transition map and UI design from psd file
  • Implemented support a localization in the application with Lean Localization framework
  • Published to Apple AppStore, Google Play Market

App description
The Viking village Kaup is a reconstruction of an early medieval settlement based on the archeology of Prussian and Scandinavian settlements and forts, which was built like a Danish military fortress of the 9th-11th centuries.
